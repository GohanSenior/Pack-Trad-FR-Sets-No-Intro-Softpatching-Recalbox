\ DESCRIPTION
 \_____________


C'est un petit jeu assez fun avec lequel vous ne devriez pas vous ennuyer. Il est basé sur le modèle Tetris mais avec 
quelques différences notables. Vous avez en effet le contrôle de vaisseaux et vous devrez former des figures bien précises
avec vos tirs :

                                                          
 ____    ________              ___________________                             
|    |  |        |  ________  |                   |  _________________________ 
|    |  |        | |        | |                   | |                         |
|    |  |        | |________| |                   | |_________________________|
|____|  |________|            |___________________|                            
                                                                            
Evidemment si les blocs atteignent le bas de l'écran s'en est fini pour vous !
Bref il faut très vite former des blocs pour les éliminer et passer au suivant. J'adore cela va vite et en plus avec un
 peu de rélfexion les blocs s'enchainent rapidemment.


\ COMMENT APPLIQUER LE PATCH
 \___________________________

ATTENTION !!! Le patch est à appliquer sur la rom quarth (u) [!].gb
              La rom doit faire 65 536 octets.
	      Les autres roms (si elles existent) n'ont pas été testées. Il est donc possible que ça ne marche pas.

Pour appliquer le patch de traduction, il faut tout d'abord récupérer un outil pour patcher.
Il en existe plusieurs que vous pourrez trouver sur le site de la TRAF (http://traf.romhack.org).
Il suffit ensuite généralement de sélectionner la rom puis le fichier ips et de valider
(Certains outils peuvent créer une sauvegarde de la rom avant de patcher).
Voilà maintenant votre rom avec les modifications apportées par notre traduction.
