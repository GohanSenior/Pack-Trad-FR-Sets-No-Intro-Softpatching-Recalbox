Patch traduisait "Dragon Ball Z - Super Gokuuden - Totsugeki-Hen (Super Famicom) en français.

Romhack de Neige, GFX de Lestat/m0nsieurL et Hiei- pour le reste.

L'archive contient deux patchs IPS. L'un est le patch de traduction, l'autre est un patch permettant d'activer le debug mode. Il peut être appliqué avant ou après
le patch de traduction, cela ne change rien (il se contente de remplacer les deux octets "8004" à l'addresse 0x61 en "EAEA3 pour activer le debug mode).

La rom source à utiliser est celle du groupe No-Intro intitulée : "Dragon Ball Z - Super Gokuu Den - Totsugeki Hen (Japan).sfc". Checksum : "D531289B".

Voici quelques infos sur certains "labyrinthes" du jeu (dans l'ordre du jeu) :

Ville de Fangas (Gang des lapins) :
1, 2 : Magasin de capsules
2, 3 : Magasin de nourriture
1, 2, 2, 2, 2 : Magasin de vêtements
2, 1, 3 : Station Service (fin du "labyrinthe")

Château de Pilaf :
1, 1, 1, 1, 1, 1, 1, 2.

La recherche d'une fille pour Kame-Sennin :

Le "labyrinthe" change visiblement à chaque partie. En gros, il faut ramener une première personne au hasard, et ensuite essayer un peu
tous les choix jusqu'à tomber sur la bonne (avec les noms des différents choix, vous pouvez facilement repérer les choix déjà essayés).

Après l'apparition de Krilin, toujours choisir le choix "Aller voir ailleurs" jusqu'à tomber sur Lunch.

Pendant le déplacement vers Lunch, appuyer sur X pour consulter Krilin (obligatoire pour débloquer l'événement66, requis pour débloquer
l'événement 140, qui active le cross-over avec Dr Slump).

Recherche de la pierre :
1, 1, 1, 1, 1 : Krillin
3, 2, 3 : Ptéranadon
2, 1 : Krillin
2, 3, 1, 1, 2, 1 : Krillin
1, 1 : Ptéranadon
1, 1 : Ptéranadon
2 : Pierre

QG de l'armée du RR :
1 : Entrer de front
1 : Droite
(Le dernier choix ne semble pas important, les trois choix ramènent au même endroit)

Trouver le poisson de Yajirobe : 
2, 2, 3, 3.

Note : Pour obtenir la technique cachée (et donc avoir 45/45 techniques), lors du cross-over avec Dr Slump, il faut répondre le choix 3 lors de l'apparition d'Arale.
La technique se débloque vers la fin du jeu, si vous choississez de boire l'élixir divin (événement 157).
