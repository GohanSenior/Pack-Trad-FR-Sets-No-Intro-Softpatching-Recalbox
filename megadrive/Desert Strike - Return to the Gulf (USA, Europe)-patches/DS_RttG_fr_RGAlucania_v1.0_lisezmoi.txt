1.  L'HISTOIRE DE DESERT STRIKE - RETURN IN THE GOLF
----------------------------------------------------------------------------------------------

L'histoire prend place un an après la guerre du Golfe alors qu'un général tyrannique menace le Moyen-Orient et les Etats-Unis. Le président vous charge d'abattre le général Kilbaba et ses terroristes, avant qu'ils n'engagent un conflit nucléaire mondial. Vous pilotez un hélicoptère Apache et vos objectifs sont divers : porter secours, détruire des objectifs, détourner des missiles...

----------------------------------------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------------------------------------

Normalement, l'archive ZIP doit contenir :

[MD]DesertStrikeFR.ips  - Le patch IPS de Désert Strike
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un le Patcheur Lunar IPS
Ce patch est à utiliser avec la rom Desert Strike - Return to the Gulf (USA, Europe).md
