1.  L'HISTOIRE DE RETURN OF THE NINJA
----------------------------------------------------------------

Return of the Ninja est la suite du Shadow of the Ninja de la
NES (connu aussi sous le nom Blue Shadow).
Comme son aîné, le titre s'inspire de Ninja Gaiden et propose
enfin un bon jeu de ninja sur la petite portable de Nintendo.

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir:
ReturnOfTheNinjaFr.ips  - Le patch IPS de Return of the Ninja
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
supportant le format IPS (patcheur Terminus exemple).
Ce patch est à utiliser avec la Rom NO-INTRO
Return of the Ninja (Europe).gb
