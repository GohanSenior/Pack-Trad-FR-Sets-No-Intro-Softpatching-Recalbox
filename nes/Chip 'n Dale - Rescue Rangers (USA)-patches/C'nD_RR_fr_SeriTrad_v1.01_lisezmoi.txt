Introduction
------------

Tic et Tac : Les Rangers du risque est un jeu de plate-formes sur NES dans lequel vous incarnez le célèbre duo de Disney. Votre but est de retrouver un chat, mais une autre sombre menace plane sur vos enquêtes.
A un ou deux joueurs, vous parcourerez donc divers tableaux afin de déjouer les plans machiavéliques de vos adversaires. Le soft est en soi très facile.

Contenu
-------
Vous trouverez ici la patch de Tic et Tac : Les Rangers du risque en Français, traduite à partir de la version anglaise du soft. Le texte se veut le plus fidèle possible à cette version. 

L'écran-titre francisé a été inspiré de celui de FlashPV.

Si vous êtes bloqués dans le jeu, l'excellent site GameFaqs recense
toutes les techniques pour venir à bout du soft.

Bugs Connus
-----------
Aucun bug connu n'a filé entre les mains de nos bêta-testeurs. La présente version du patch a été testée avec Nestopia v1.42.
