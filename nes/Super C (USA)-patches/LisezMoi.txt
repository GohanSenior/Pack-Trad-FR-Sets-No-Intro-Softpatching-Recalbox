1.  L'HISTOIRE DE SUPER  C
----------------------------------------------------------------

Super C (connu aussi sous le nom Super Contra) est la suite de
Contra. Mis à part le scénario qui est vraiment basique, tout a été
amélioré dans cette suite, à noter l'apparition de stages avec une
vue en hauteur que l'on retrouvera plus tard (mais avec le fameux
mode 7) dans Contra 3, sa suite sur Super Nintendo. 

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir :

SuperCFr.ips  - Le patch IPS de Super C
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
(le Patcheur Terminus par exemple).
Ce patch est à utiliser avec la rom Super C (U)
