1.  L'HISTOIRE DE CYBERNOID - THE FIGHTING MACHINE
----------------------------------------------------------------

Initialement sorti sur ZX Spectrum, Cybernoid aura connu pas mal
de portages dont cette version NES assez sympathique et surtout
la seule version console à ce jour.
C'est un shoot-them -up avec un aspect stratégique puisqu'il ne
faut surtout pas foncer dans le tas ou c'est la mort assurée.
Le jeu est très difficile, si vous aimmez les défis, c'est pour
vous!

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir:
CybernoidFr.ips  - Le patch IPS de Cybernoid
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
supportant le format IPS (patcheur Terminus exemple).
Ce patch est à utiliser avec la Rom NO-INTRO
Cybernoid - The Fighting Machine (USA).nes
