1.  L'HISTOIRE DE OUT RUN EUROPA
----------------------------------------------------------------

Out Run ayant été un énorme succès, certains ont voulu surfer sur
la vague et US Gold a donc développé une suite qui sortira tout
d'abord sur les ordinateurs de l'époque, puis sur les consoles de

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir:
OutRunEuropaFR.bps  - Le patch IPS de Out Run Europa
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
supportant le format IPS (patcheur Terminus exemple).
Ce patch est à utiliser avec la Rom NO-INTRO
Out Run Europa (Europe).sms
