1.  L'HISTOIRE D'OUT RUN
----------------------------------------------------------------

Titre emblématique du System 16 de Sega, Out Run proposait une
réalisation incroyable pour l'époque avec des musiques qui sont
restées dans les mémoires. Cette version Master System est une
excellente conversion qui exploite à fond les capacités de la
petite 8 bits de Sega.

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir:
OutRunSmsFr.ips  - Le patch IPS d'Out Run
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
(le Patcheur Terminus par exemple).
Ce patch est à utiliser avec la rom NO-INTRO Out Run (World).sms
