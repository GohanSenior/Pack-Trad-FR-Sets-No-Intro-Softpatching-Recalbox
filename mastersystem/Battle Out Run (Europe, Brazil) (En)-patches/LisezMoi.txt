1.  L'HISTOIRE DE BATTLE OUT RUN
----------------------------------------------------------------

Surfant sur le succès de Chase HQ, Sega propose ce spin-off de
son fameux jeu de course, Battle Out Run vous met toujours au
volant d'une Ferrari mais vous devez cette fois arrêter des
criminels en les percutant avec votre bolide. Petite originalité,
vous pourrez acheter des améliorations pour votre véhicule.
Un bon petit jeu avec une très bonne réalisation. 

----------------------------------------------------------------
2.  CONTENU DE L'ARCHIVE ET UTILISATION
----------------------------------------------------------------

Normalement, l'archive ZIP doit contenir:
BattleOutRunFr.bps  - Le patch IPS de Battle Out Run
LisezMoi.txt - Le fichier que vous êtes en train de lire

Pour utiliser cette traduction, il faut utiliser un PATCHEUR
supportant le format IPS (patcheur Terminus exemple).
Ce patch est à utiliser avec la Rom NO-INTRO
Battle Out Run (Europe).sms
